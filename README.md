# Exploratory Data Analysis Projects

This repository contains EDA performed on multiple datasets.

## 1. Titanic Dataset
- Performed data cleaning and handling missing values
- Analyzed survival patterns based on age, gender, class
- Used visualization to identify key insights

## 2. Smartphone Usage & Addiction
- Studied user behavior and screen time patterns
- Identified factors influencing addiction levels
- Visualized trends using graphs

## 3. Telecom Churn Dataset
- Analyzed customer churn behavior
- Found key factors affecting customer retention
- Used EDA techniques to derive insights
